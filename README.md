Project Overview
This project involves the design and development of a responsive multi-page website for the SPCA South Africa organisation. The website is built using HTML, CSS, and JavaScript and is intended to raise awareness about animal welfare, promote animal adoption, encourage donations, and support volunteer engagement.

The website is structured to ensure easy navigation, clear presentation of information, and a user-friendly experience. Ethical web development practices were followed, including proper content sourcing, referencing, and accessibility considerations.

---

Organisation Overview
The chosen organisation is the SPCA South Africa, a non-profit organisation dedicated to preventing cruelty to animals, rescuing abused and neglected animals, and promoting responsible pet ownership across South Africa.

---

Website Goals and Objectives
The main goals of the website are:

* To raise awareness about animal cruelty and prevention
* To promote animal adoption and rehoming services
* To encourage donations to support SPCA operations
* To increase volunteer participation
* To provide accessible and structured information about SPCA services

Key Performance Indicators (KPIs):

* Number of adoption enquiries submitted
* Number of volunteer sign-ups
* Donation enquiry interactions
* Website traffic and engagement time
* Form submission rates

---

Key Features and Functionality

* Multi-page navigation system across all pages
* Homepage with featured rescue stories and call-to-action sections
* About page with mission, vision, and organisational purpose
* Services page outlining rescue, adoption, veterinary care, and outreach programmes
* Enquiry form for adoption, donations, and volunteering
* Contact page with emergency hotline, general enquiries, and branch information
* Structured image-based storytelling for user engagement

Ethical Considerations

* All content is sourced from verified animal welfare organisations
* Images are obtained from free-use or official sources with proper referencing
* Content avoids exploitation of animal suffering and maintains respectful tone
* POPIA compliance is considered for all form data collection
* Transparency in donation and adoption messaging is maintained
* Proper Harvard referencing is applied throughout research and documentation

---

Accessibility Considerations

* All images include descriptive alternative text
* Navigation is clear and consistent across pages
* Content is structured using semantic HTML
* Website design is planned to support mobile responsiveness
* Readability is prioritised through simple layout structure
